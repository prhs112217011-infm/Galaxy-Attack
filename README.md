# Galaxy Attack

本計畫隸屬於開源遊戲組織 [Amegma](https://github.com/Amegma)，其靈感源自 Atari 街機遊戲《太空侵略者》(Space Invaders)。

[遊戲示範影片](https://youtu.be/ejA89NtTH2k)

##使用的技術

- Python

- 框架 - PyGame

- 其他模組 - time、os、sys、ctypes 等

## 設定指南

### 前提條件

- 已安裝 Python：`Python 版本 > 3.10`

- 已安裝 Pygame：`Pygame 版本 > 1.9.5`

### 運行遊戲

```

python <file_path>/main.py

```

### 建立發布版本

```

pyinstaller.exe .\main.spec

```
